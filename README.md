# 원티드 프리온보딩 프론트엔드 4팀 - 2주차 과제

> 원티드 프리온보딩 과제 2주차 (칸반보드)
>
> 기간 : 2023년 1월 3일 ~ 2023년 1월 6일

## 실행방법

#### npm install

#### npm start






###  localStorage
* 생성한 이슈들을 관리하기 위해 로컬스토리지를 사용했습니다. 
* 로컬스토리지를 어떻게 사용하는지에 집중하기 보다 비즈니스 로직 구현에 집중할 수 있도록 recoil-persist를 사용했습니다.
